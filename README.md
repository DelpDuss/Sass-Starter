# Starter project

Simple starter kit for testing stuff.

## Installation & Usage

```sh
git clone https://github.com/DelpDuss/Sass-starter.git projectName
cd projectName
rm -rf .git
npm install
npm run dev
```

## Inspiration & Ressources

- [HTML5 Boilerplate](https://github.com/h5bp/html5-boilerplate)
